Introdução ao Python

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Criada por Guido van Rossum em 1991, é conhecida por sua simplicidade e versatilidade.

Aprenda Python

Sintaxe Básica

A sintaxe do Python é simples e clara. Aqui estão alguns exemplos básicos:


# Isto é um comentário
print("Olá, Mundo!")

# Variáveis e tipos de dados
x = 5
y = "Texto"
z = 3.14

# Estruturas de controlo
if x > 0:
    print("x é positivo")
elif x < 0:
    print("x é negativo")
else:
    print("x é zero")

Estruturas de Dados

Python oferece várias estruturas de dados embutidas, incluindo:

  • Listas: Coleções ordenadas e mutáveis
  • Tuplos: Coleções ordenadas e imutáveis
  • Dicionários: Coleções não ordenadas de pares chave-valor
  • Conjuntos: Coleções não ordenadas de elementos únicos

Funções

As funções em Python são definidas usando a palavra-chave 'def'. Elas podem ter argumentos posicionais, argumentos nomeados e valores padrão.


def saudacao(nome, saudacao="Olá"):
    return f"{saudacao}, {nome}!"

print(saudacao("Maria"))  # Saída: Olá, Maria!
print(saudacao("João", "Bom dia"))  # Saída: Bom dia, João!

Módulos

Os módulos permitem organizar o código Python em ficheiros separados. Você pode importar módulos inteiros ou funções específicas de módulos.


# Importar um módulo inteiro
import math

# Importar funções específicas
from random import randint, choice

# Usar funções importadas
print(math.pi)
print(randint(1, 10))

Pip e PyPI

Pip é o gerenciador de pacotes para Python, usado para instalar e gerenciar pacotes adicionais que não fazem parte da biblioteca padrão. PyPI (Python Package Index) é o repositório de software para a comunidade Python.


# Instalar um pacote usando pip
pip install nome_do_pacote
